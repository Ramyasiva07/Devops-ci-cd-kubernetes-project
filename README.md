🚀 DevOps CI/CD Pipeline with Kubernetes

📌 Overview

Implemented a CI/CD pipeline to deploy a Python application using Docker, Jenkins, and Kubernetes.

🔧 Tools Used

- Docker
- Jenkins
- Kubernetes (Minikube)
- GitHub

⚙️ Workflow

1. Code pushed to GitHub
2. Jenkins triggers pipeline
3. Docker image built
4. Deployment updated in Kubernetes

🌍 Access Application

minikube service ramya-service

📦 Kubernetes Components

- Deployment (app replicas)
- Service (NodePort exposure)

🎯 Key Learnings

- CI/CD automation
- Containerization
- Kubernetes deployment strategy
